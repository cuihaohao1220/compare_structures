# compare_structures Python 包

这是 `compare_structures` 的 Python 版本。

## 安装

```bash
pip install -e .
```

## 使用

```python
from compare_structures_py import compare_structures

differences = compare_structures(
    origin_data={"a": 1},
    current_data={"a": 2}
)
```

更多详细信息请参考根目录的 [README.md](../README.md)。
